# Bash Scripting Journal for DevOps

A hands-on Bash scripting learning journal created as part of my DevOps preparation journey.

This repository contains daily labs, scripts, notes, assignments, and interview-focused exercises covering Bash scripting fundamentals and real-world DevOps automation tasks.

The goal of this repository is to:

* Practice DevOps automation
* Prepare for DevOps and Cloud interviews
* Maintain a structured learning journal
* Create a portfolio of Bash projects
